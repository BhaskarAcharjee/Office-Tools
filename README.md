# 🚀 Office Tools Suite

> A collection of powerful internal productivity tools for developers and analysts.

🔗 **Live Demo:**  
https://bhaskaracharjee.github.io/Office-Tools/

---

## ✨ Overview

Office Tools Suite is a lightweight web-based utility platform designed to simplify repetitive office and development tasks.

It focuses on:
- Improving code analysis efficiency
- Generating structured API evidence
- Creating clear and consistent specifications

Built with simplicity, speed, and practical usage in mind.

---

## 🛠 Available Tools

### 1️⃣ If-Else Extractor
Extract specific nested `if-else` logic blocks from large Java methods using a keyword.

**Use Cases:**
- Debugging complex legacy code
- Understanding nested decision trees
- Isolating feature-specific logic

🔗 `/if-else`

---

### 2️⃣ API Evidence Generator
Generate structured API request and response documentation for audit and validation.

**Use Cases:**
- QA documentation
- Client deliverables
- Compliance and audit evidence

🔗 `/api-evidence`

---

### 3️⃣ Specification Generator
Create clean functional and technical specifications from structured inputs.

**Use Cases:**
- Requirement documentation
- Feature planning
- Technical design discussions

🔗 `/spec-generator`
